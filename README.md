Quiz em HTML e CSS 🎮

Este é um projeto simples de Quiz interativo desenvolvido utilizando apenas HTML e CSS.
O objetivo é praticar conceitos de estruturação de páginas e estilização, além de oferecer uma experiência divertida ao usuário.

🚀 Funcionalidades

Perguntas de múltipla escolha.

Design responsivo e agradável.

Feedback visual para respostas corretas e incorretas.

🛠 Tecnologias utilizadas

HTML5

CSS3
